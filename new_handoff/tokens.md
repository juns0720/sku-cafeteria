### 자간(letter-spacing) 룰

- **800 굵기 + 16px 이상**: -0.3 ~ -0.9 (제목)
- **700 굵기 + 14~15px**: -0.2 ~ -0.3
- **500~600 + 11~13px**: 0 (또는 -0.2 강조 시)

## 그림자

```css
/* 모바일 프레임 */
box-shadow:
  0 2px 8px rgba(0,0,0,0.06),
  0 12px 40px rgba(0,0,0,0.08);

/* 슬라이더 thumb (사용 시) */
box-shadow: 0 2px 6px rgba(255,107,92,0.3);
```

웹앱 화면 내부 카드들은 그림자를 거의 사용하지 않습니다. 헤어라인(`1px solid #F2F4F6`)으로 분리.

## 모바일 프레임

```
width: 375px
height: 760px
border-radius: 32px
overflow: hidden
display: flex; flex-direction: column;
```
